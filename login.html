Aquí te explico los pasos: Clona el repositorio o accede al proyecto Si aún no has clonado el repositorio, usa este comando: git clone Luego, navega al directorio del proyecto: cd Si ya tienes el repositorio clonado, simplemente accede al directorio. Asegúrate de estar en la rama correcta Comprueba en qué rama estás trabajando: git branch Si necesitas cambiar de rama, usa: git checkout Si la rama no existe localmente, puedes crearla y cambiar a ella: git checkout -b Realiza los cambios Edita los archivos necesarios en tu proyecto. Añade los cambios al área de preparación Verifica qué archivos se han modificado: git status Luego, añade los archivos que deseas incluir en el commit: git add O añade todos los archivos modificados: git add .
